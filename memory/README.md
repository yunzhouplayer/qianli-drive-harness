# Memory 长期记忆

本目录用于沉淀项目长期记忆，包括平台共识、架构决策、经验复盘和 Agent 使用反馈。

Memory 与 Knowledge 的区别：

- Memory 记录项目演进过程中形成的共识、决策和经验。
- Knowledge 记录相对稳定的业务事实、测试规则、状态机和数据质量规则。

## 子目录

- `project/`：平台定位、模块边界、长期共识。
- `decisions/`：架构决策记录 ADR。
- `lessons/`：测试开发、仿真、Agent 协作经验复盘。
- `agent-feedback/`：Agent 产物采纳率、失败模式、Prompt/Skill 改进建议。

## 禁止存储

- 密钥、Token、生产账号。
- 未脱敏用户数据。
- 真实乘客敏感信息。
- 真实车辆敏感轨迹。
- 不应共享的个人偏好或组织敏感信息。
