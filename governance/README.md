# Governance 平台治理

本目录用于沉淀 AI 产物、测试数据、仿真数据、安全合规和发布门禁治理规则。

## 子目录

- `ai-output/`：AI 生成代码、用例、报告、建议的审核和准入规则。
- `data-security/`：敏感数据、脱敏、权限、密钥、生产数据隔离规则。
- `simulation-data/`：仿真数据真实性、偏差、污染、可追溯性治理。
- `release-gates/`：发布准入、质量门禁、风险升级和人工复核规则。

## 原则

- AI 提升效率，但不能绕过确定性校验。
- 生产数据、未脱敏用户数据、密钥和敏感轨迹不得进入仓库。
- 仿真数据必须可追溯来源、生成规则和适用范围。
- 高风险 Agent 产物必须经过人工复核和 Harness 质量门禁。
